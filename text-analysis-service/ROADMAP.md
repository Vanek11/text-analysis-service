# Roadmap: План развития сервиса анализа английского текста

## Версия 1.1.0 (в разработке)

### Первый этап (High Priority) — 1-2 месяца

#### ✅ 1. Классификация наречий (Semantic + Morphological)
- [ ] Семантическая классификация (Manner, Time, Place, Frequency, Degree, Sentence)
- [ ] Морфологическая классификация (Simple, Derived, Compound, Phrasal)
- [ ] Вкладка "Наречия" с таблицей и фильтрами
- [ ] Визуализация связей наречий с модифицируемыми словами

#### ✅ 2. Анализ причастий
- [ ] Определение Present Participles (-ing формы)
- [ ] Определение Past Participles
- [ ] Роли причастий (Progressive, Perfect, Passive, Adjective, Absolute, Gerund)
- [ ] Визуализация в дереве зависимостей

#### ✅ 3. Фразовые и модальные глаголы
- [ ] Определение фразовых глаголов
- [ ] Классификация модальных глаголов
- [ ] Анализ вспомогательных глаголов
- [ ] Расширенная таблица глаголов

#### ✅ 4. Проверка грамматики
- [ ] Subject-Verb Agreement проверка
- [ ] Article Usage проверка
- [ ] Verb Tense Consistency
- [ ] Вкладка "Проверка грамматики" с ошибками и предупреждениями

#### ✅ 5. Улучшенная визуализация
- [ ] Pie Chart для распределения POS-тегов
- [ ] Heat Map распределения частей речи
- [ ] Улучшенное дерево зависимостей с фильтрацией

## Версия 1.2.0 (планируется)

### Второй этап (Medium Priority) — 2-3 месяца

#### 6. Классификация прилагательных
- Классификация по типу (Descriptive, Quantitative, Demonstrative, etc.)
- Анализ степеней сравнения (Positive, Comparative, Superlative)
- Правила образования степеней
- Вкладка "Прилагательные"

#### 7. Грамматические конструкции
- Временные конструкции (все 12 времен)
- Условные предложения (Type 1, 2, 3)
- Косвенная речь
- Пассивный залог
- Вкладка "Грамматические конструкции"

#### 8. Анализ предлогов
- Классификация предлогов (Time, Place, Direction, Agent, Instrument, Purpose)
- Анализ предложных фраз
- Вложенные предложные фразы
- Вкладка "Предлоги"

#### 9. Метрики синтаксической сложности
- Средняя длина предложения
- Средняя глубина дерева зависимостей
- Коэффициент сложности
- Лексическое разнообразие (TTR)
- Индекс Flesch-Kincaid
- Определение уровня (A1-C2)
- Вкладка "Анализ сложности"

#### 10. Новые диаграммы
- Radar Chart (диаграмма лепестков)
- Sankey Diagram
- Timeline для хронологии событий
- Word Cloud

---

## Версия 1.3.0+ (планируется)

### Третий этап (Nice-to-Have) — 3+ месяца

#### 11. Интерактивное обучение
- Fill-in-the-blank упражнения
- Multiple Choice вопросы
- Matching упражнения
- Reordering упражнения
- Hover Tooltips с объяснениями
- Вкладка "Обучение"

#### 12. Расширенная статистика текста
- Основные метрики (слова, символы, предложения)
- Лексическая статистика
- Word Frequency Table
- Индексы читаемости
- Вкладка "Статистика"

#### 13. История и сохранение
- Сохранение последних анализов (localStorage)
- Сравнение анализов
- Закладки для текстов
- Экспорт результатов (JSON, CSV, PDF, DOCX)

#### 14. Учетные записи пользователей
- Регистрация и вход
- Облачное сохранение прогресса
- Статистика использования
- Синхронизация между устройствами

#### 15. Многоязычность
- Британский vs Американский английский
- Поддержка других языков (испанский, французский, немецкий)
